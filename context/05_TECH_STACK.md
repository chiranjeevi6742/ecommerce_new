# 05. Technology Stack

## Frontend
| Technology | Version | Purpose |
| :--- | :--- | :--- |
| **Next.js** | 14.x (App Router) | React Meta-Framework for SSR/CSR |
| **React** | 18.x | UI Component Library |
| **TypeScript** | 5.x | Static Typing |
| **Tailwind CSS** | 3.x / 4.x | Styling |
| **Framer Motion** | Latest | Animations (Hero, Drawers) |
| **Lucide React** | Latest | Icons |
| **Supabase JS** | Latest | Auth Client & Storage Client |

| Technology | Version | Purpose |
| :--- | :--- | :--- |
| **Python** | 3.10+ | Runtime Language |
| **FastAPI** | Latest | Web Framework (High Performance) |
| **Uvicorn** | Latest | ASGI Server |
| **Pydantic** | Latest | Data Validation & Schemas |
| **Psycopg2** | Latest | PostgreSQL Adapter |
| **Python-Dotenv** | Latest | Environment Variable Management |
