# InstiShop - Project Context

## Welcome to the InstiShop Project Context

This directory contains comprehensive documentation designed to provide any AI agent (or human developer) with a complete understanding of the **InstiShop** e-commerce platform.

**InstiShop** is a full-stack e-commerce application designed specifically for campus/institute environments, allowing students to buy merchandise, course materials, and essentials with same-day delivery.

## Purpose of this Context
The goal of this documentation is to enable seamless context switching and onboarding. By reading these files, an agent should be able to:
1.  Understand the business goals and user personas.
2.  Navigate the codebase (Frontend & Backend) effectively.
3.  Implement new features following established patterns.
4.  Debug issues with knowledge of the data flow and architecture.
